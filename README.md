# Saanvi Agrawal - Personal Website

A beautiful, modern, and responsive personal website showcasing professional experience, skills, achievements, and personal interests with a stunning baby blue color scheme.

## Features

- **Modern Design**: Clean, aesthetic design with smooth animations and baby blue gradients
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Smooth scrolling, hover effects, form validation, and image sliders
- **Professional Sections**: Hero, About, Experience, Activities, Events & Conferences, Skills, Hobbies, and Contact sections
- **Mobile-Friendly Navigation**: Hamburger menu for mobile devices
- **Smooth Animations**: Fade-in animations, parallax effects, and interactive sliders
- **Image Sliders**: Interactive sliders for events and conferences with navigation controls
- **Contact Form**: Functional contact form that opens email client with pre-filled message

## Customization Guide

### Personal Information
Update the following in `index.html`:

1. **Name and Title**: Update the hero section with your name and title
2. **Contact Information**: Update email, location, LinkedIn, and GitHub in the contact section
3. **About Section**: Modify the description to match your background
4. **Experience**: Update job titles, companies, dates, and achievements
5. **Activities**: Update leadership roles, descriptions, and organization links
6. **Events**: Add or modify conference experiences and photos
7. **Skills**: Add or remove skills in the skills section
8. **Hobbies**: Update hobbies, descriptions, and associated photos

### Styling
Modify `styles.css` to customize:

1. **Colors**: Change the primary color scheme (currently baby blue: #87CEEB, #B0E0E6, #E0F6FF, #F0F8FF)
2. **Fonts**: Update font families and sizes
3. **Layout**: Adjust spacing, padding, and grid layouts
4. **Animations**: Modify transition effects and timing
5. **Sliders**: Customize slider appearance and behavior

### Functionality
Update `script.js` to customize:

1. **Form Handling**: Modify the contact form submission logic
2. **Animations**: Adjust animation timing and effects
3. **Interactive Elements**: Customize hover effects and interactions
4. **Sliders**: Modify slider functionality and controls
